# TaskFlow - Django Authentication System

## Overview

TaskFlow is a Django-based web application that implements a complete user authentication system. The project includes user registration, login, logout, role-based access control, and password reset functionality.

This project was developed as part of the CodeVeda Advanced Level Internship Program.

---

## Features

* User Registration
* User Login
* User Logout
* Secure Password Hashing
* Role-Based Access Control
* Admin Dashboard
* Password Reset Functionality
* SQLite Database Integration
* Session Management
* Form Validation

## Installation

1. Clone the repository

git clone <repository-url>

2. Navigate to the project directory

cd TaskFlow

3. Create virtual environment

python -m venv venv

4. Activate virtual environment

Windows:

venv\Scripts\activate

5. Install dependencies

pip install django

6. Run migrations

python manage.py migrate

7. Start server

python manage.py runserver
